## Quick Start — NVIDIA Windows

1. Install Python 3.11 or 3.12.
   - During installation, tick **Add Python to PATH**.
2. Download this repository as a ZIP.
3. Extract the ZIP.
4. Double-click `run_windows_nvidia.bat`.
5. Wait for first-time setup to finish.
6. The app will open automatically.

First launch may take several minutes because the script creates a local `.venv` folder and installs CUDA PyTorch, Whisper, PySide6, and FFmpeg support.

After that, future launches are much faster.
